🔍 Data Authenticity Principles

This dashboard strictly follows a source-priority hierarchy. We never fabricate or interpolate missing data. Where data is unavailable, we display an explicit "N/A" with a reason. Each data point is tagged with a confidence level (HIGH / MEDIUM / LOW) and linked to its raw snapshot.

Source Priority: (1) Official REST/JSON APIs → (2) Official CSV/XLSX/open data portals → (3) Official HTML pages → (4) Official PDFs with table extraction → (5) Trusted secondary sources (clearly labeled).

⚠ We respect robots.txt and all source terms of service. We do not bypass paywalls, anti-bot systems, or rate limits. All scrapers run on a 10-second crawl-delay where required.
